## Flatpak

- VS Code

```sh
flatpak install com.visualstudio.code
```

- Discord

```sh

```

- Slack

```sh
flatpak install com.slack
```

- Neovim

```sh
flatpak install neovim
```

- Brave

```sh
flatpak install com.brave
```

- Spotify

```sh
flatpak install spotify
```
