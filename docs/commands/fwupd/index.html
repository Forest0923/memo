fwupd

Install

  • Arch
  • Ubuntu
```sh
sudo pacman -S fwupd
```
```sh
sudo apt install fwupd
```

Usage

  • Refresh data from server:
fwupdmgr refresh
  • Check device for updates:
fwupdmgr get-updates
  • Update and install:
fwupdmgr update

Troubleshooting

  • If the update does not apply, follow the steps below.
  1. Download .cab file
  2. Install:
fwupdmgr install [path-to-.cab]
