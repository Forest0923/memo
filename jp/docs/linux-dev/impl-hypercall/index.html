    ハイパーコールの実装 #

    QEMU/KVM にハイパーコールを追加するときのメモです．

    環境 #

    • CPU: x86_64
    • Hypervisor: Linux KVM 5.8.13
    • Guest: Linux 5.8.13

    チュートリアル #

    1. KVM 側のハイパーコールの追加 #

    include/uapi/linux/kvm_para.h を修正し，新しいハイパーコール番号を登録します．

+#define KVM_HC_HELLO_HYPERCALL1	13

    ハイパーコールは VMCALL という VM Exit を発生させ，その exit ハンドラは最終的に arch/x86/kvm/x86.c にある kvm_emulate_hypercall() を呼び出します．その中の switch 文を次のように書き換えることでハイパーコールを追加できます．

     		kvm_sched_yield(vcpu->kvm, a0);
 		ret = 0;
 		break;
+	case KVM_HC_HELLO_HYPERCALL0:
+		trace_printk("Hello world!\n");
+		break;
+	case KVM_HC_HELLO_HYPERCALL1:
+		trace_printk("a0 = %d\n", a0);
+		break;
 	default:
 		ret = -KVM_ENOSYS;
 		break;

    2. ゲスト側からのハイパーコールの呼び出し #

    ハイパーコールの呼び出しは次のように kvm_hypercall0() などを使用して行います．引数の最大は4までです．

    #include <linux/kernel.h>
#include <linux/syscalls.h>
#include <uapi/linux/kvm_para.h>
#include "my_syscalls.h"

SYSCALL_DEFINE0(hello_hypercall0)
{
    kvm_hypercall0(KVM_HC_HELLO_HYPERCALL0);
    return 0;
}

SYSCALL_DEFINE1(hello_hypercall1, int, arg1)
{
    kvm_hypercall1(KVM_HC_HELLO_HYPERCALL1, arg1);
    return 0;
}

    ユーザ空間から直接呼び出すことはできないので kvm_hypercallk() を呼び出すためのシステムコールを追加する必要があります．以下のパッチは単純なハイパーコールを呼び出すシステムコールを追加するものです．

    3. テスト #

    ゲストからハイパーコールを呼ぶためのシステムコールを呼ぶプログラム：

    #include <stdio.h>
#include <sys/syscall.h>

#define HELLO_HYPERCALL0    440
#define HELLO_HYPERCALL1    441

int main(int argc, char **argv)
{
    syscall(440);
    syscall(441, 123);
    return 0;
}
