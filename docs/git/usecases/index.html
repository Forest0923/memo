    Usecases of Git #

    Cancel git add # 

    git reset
# or
git reset <file>

    Cancel git commit # 

    # Reset contents 
git reset <tag> --hard

# Reset commit and keep changes
git reset <tag> --soft

    View previous commit # 

    git checkout <tag>
